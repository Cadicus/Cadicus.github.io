3D Scanner Input FAQ

What types of scans work with Cadicus?

STL (Stereolithographic data files .stl) scans of below-knee residual limbs.


Do all scans work?

We prefer a manifold mesh if possible, but will do the best we can to clean up the mesh and make it manifold before processing.


What is a manifold mesh?

Manifold is a mathematical term that means there are no duplicate or overlapping triangles or points in the mesh. The mesh exists as one single surface with no artifacts that would cause processing errors. Link to wikipedia for manifold mesh.


What types of tools can be used to process the mesh?

Open-source software like meshmixer or blender should be sufficient for cleaning up the mesh, but software that comes packaged with scanners may also offer this functionality.
